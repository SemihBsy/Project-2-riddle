# Riddler App

### Description of the App:
Riddler APP allows users to add, create, edit and delete riddles. Users also be able to create their own list of riddles.

### Technologies Used
1.Node
2.Express
3.HTML
4.CSS
5.JavaScript
6.EJS
7.MongoDB
